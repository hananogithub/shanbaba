# Shan Baba - 多国籍料理レストラン

Shan Babaの公式ウェブサイトです。ハラル認証を受けた多国籍料理を提供するレストランのモダンでレスポンシブなウェブサイトです。

## 特徴

- **レスポンシブデザイン**: デスクトップ、タブレット、スマートフォンに対応
- **モダンなUI/UX**: 美しいアニメーションと直感的なナビゲーション
- **ハラル認証表示**: ハラル認証バッジで安心感を提供
- **多言語対応**: 日本語での完全なコンテンツ
- **インタラクティブ機能**: 
  - メニューカテゴリーフィルタリング
  - ギャラリーライトボックス
  - スムーススクロール
  - 予約フォーム

## セクション

### 1. ホーム
- ヒーローセクション with レストランのメイン画像
- レストランの紹介とCTAボタン

### 2. 私たちについて
- レストランのストーリー
- ハラル認証、多国籍料理、心を込めた料理の特徴
- レストラン内装の画像

### 3. メニュー
- カテゴリー別メニュー表示
- 中東料理、アジア料理、ヨーロッパ料理、デザート
- 価格表示と料理の説明

### 4. ギャラリー
- 料理とレストラン空間の写真
- ライトボックス機能で拡大表示
- キーボードナビゲーション対応

### 5. お問い合わせ
- レストラン情報（住所、電話、メール、営業時間）
- 予約フォーム
- バリデーション機能付き

## 技術仕様

- **HTML5**: セマンティックマークアップ
- **CSS3**: Flexbox、Grid、アニメーション
- **JavaScript (ES6+)**: モダンなJavaScript機能
- **レスポンシブデザイン**: モバイルファーストアプローチ
- **アクセシビリティ**: キーボードナビゲーション対応

## 使用方法

1. `index.html`をブラウザで開く
2. ナビゲーションメニューで各セクションに移動
3. メニューカテゴリーをクリックして料理をフィルタリング
4. ギャラリー画像をクリックしてライトボックス表示
5. お問い合わせフォームで予約
